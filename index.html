⏱ Timelapse repeat mission — before leaving home
Do NOT edit the saved waypoint mission — adding waypoints mid-project causes sorting chaos. Create a second mission instead
Note the EV setting used on the last session — match it today
Shoot RAW (DNG) if possible — far more exposure latitude than JPEG
⏱ Timelapse repeat mission — on site consistency checks
Launch from the EXACT same point every session — use a physical marker (stake, spray paint dot)
Power on and wait 60–90 seconds before arming — GPS lock tightens significantly. Do not skip this
Verify active lens is WIDE (1x) — not tele. Check in DJI Fly before arming
Set MANUAL exposure before loading the mission — record your values each session
Adjust EV BEFORE arming — locks in at arm time. Overcast/dusk = +0.7 to +1.3 EV. Bright sun = -0.3 to -0.7 EV
Verify gimbal pitch matches your standard angle — check in DJI Fly before arming
Verify heading lock set to standard bearing for each waypoint
Calibrate gimbal if horizon looks tilted or gimbal has been bumped — 30 seconds in DJI Fly settings
Calibrate IMU only if prompted or horizon drift visible — on flat level surface
Calibrate compass only if prompted — do not over-calibrate
Confirm waypoint count matches previous sessions before arming
⏱ Timelapse repeat mission — watch for during flight
Zoomed-in frame — zoom resets to whatever was set at arm time. If one photo looks tighter, check zoom before next session
Tilted gimbal frame — shows sky and roofline instead of scene. Delete before sorting
Wrong lens active — scene noticeably wider or tighter than usual. Confirm 1x wide before next session
⏱ Timelapse repeat mission — post-flight
Review all images before importing — delete bad frames (wrong angle, zoom, very dark) now
Move session folder into Construction/ folder on NVMe
Sort new session into CarsonsSorted/ waypoint folders (sort_to_waypoints.py)
Run alignment preview: align_waypoint.py ... --preview
Delete any high-border problem frames, then run: align_waypoint.py ... --ref 22 --crop-pct 8 --exposure
